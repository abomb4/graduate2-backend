= 候选人接口

== 接口列表
- 分页复杂条件查询人才库（候选人集合）
- 根据候选人id查找候选人信息

=== 分页复杂条件查询人才库
本接口仅提供给前台页面使用。

==== URL
```
GET /myProfile/candidate/list?
```

==== 请求参数
[options="header"]
|======
| 字段 | 说明 | 类型 | 备注 | 是否必填
| pageNo | 页码 | Number | - | Y
| pageSize | 分页大小 | Number | - | Y
|======
